SourceCalc Privacy Policy

Last updated: March 13, 2026

Overview

SourceCalc is a Chrome extension that calculates profit margins for products sourced from 1688.com. We are committed to protecting your privacy.

Data Collection

We do not collect any personal data. Specifically:

  • No personal information (name, email, etc.) is collected
  • No browsing history is collected or tracked
  • No analytics or tracking scripts are used
  • No cookies are set by the extension
  • No data is sent to any server we operate

Data Storage

The extension uses Chrome's local storage API (chrome.storage.local) to save your preferences (target market, platform, shipping method, weight, quantity). This data is stored entirely on your device and is never transmitted externally.

External Requests

The extension makes a single external request to fetch the current CNY/USD exchange rate from a public API. No personal or identifying information is included in this request.

Permissions

  • activeTab — Used to detect if the current page is on 1688.com
  • storage — Used to save user preferences locally
  • Host permission (1688.com) — Used to display the calculator panel on 1688.com product pages

Third Parties

We do not sell, transfer, or share any user data with third parties.

Changes

If this privacy policy changes, we will update this page with the new date.

Contact

If you have questions about this privacy policy, please open an issue on our GitHub repository.
