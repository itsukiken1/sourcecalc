# SourceCalc - 1688 Profit Calculator Chrome Extension

Calculate your real profit margin instantly when sourcing products from 1688.com.

## What it does

When you browse any product page on 1688.com, SourceCalc automatically:
- Detects the product price from the page
- Shows a profit calculator panel on the right side
- Calculates: sourcing cost + shipping + customs + platform fees = real profit

## Features

- **Auto price detection** - reads prices directly from 1688 product pages
- **7 target markets** - USA, UK, Germany, Japan, Australia, UAE, Saudi Arabia
- **6 platforms** - Amazon FBA/FBM, Shopify, Temu, eBay, TikTok Shop
- **Sea & air freight** estimates
- **Real-time exchange rates** (CNY/USD)
- **Bilingual UI** (Chinese + English)
- Settings saved between sessions
- No data collection - all calculations run locally

## Install

1. Download or clone this repo
2. Open `chrome://extensions/` in Chrome
3. Enable **Developer Mode** (top right)
4. Click **Load Unpacked** → select this folder
5. Open any product on 1688.com
6. The profit calculator panel appears on the right
